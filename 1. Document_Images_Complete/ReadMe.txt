This folder contains all the variations(different stages of preprocessging output image) 
of the Document Images.
i.e,

1) Original_Images: 20 camera captured original document images.
2) Gray_Images    : output of Preprocessing all 20 original document images.
3) Binary_Images  : output of Preprocessing all 20 Gray document images. 
4) OCRtxt_GT      : output of Preprocessing all 20 Binary document images.
